Scrappie

Scrappie is a recipe scrapping webapp where the user can either web scrape a recipe online or upload their own recipe to the database. The name of this app is a play of the words between scrape book and web scrapping. To get the images from the recipe url, I used a web scraping library to get the HTML of the page. Then I use getElementsByTagName to get all of the img tag and after that filter out ad banners and other unwanted images. For account authentication, I utilised an library called PassportJS to enable the creation of accounts via email, facebook, twitter, or google. Data for the recipes that the user has saved are stored in a database as JSON format in a mlab (a cloud mongoDB database) and storage for the pictures are stored in Amazon S3.

Tic Tac Toe

This tic tac toe game is programmed to never lose. There are 4 main moves the computer can take. An offensive move, where the computer can determine which grid to place in order to win the game. A defensive move, where the computer can determine which grid to place in order to block to user from winning the game. The last 2 moves are predetermined moves where it will always guarentee a win based on the starting position of the user.
