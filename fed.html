Scrappie

I made this app to easily bookmark recipes online. The name is a play of the words scrape book and web scrapping. To get the images from the recipe url, I used a web scraping library to get the HTML of the page. Then I use getElementsByTagName to get all of the img tag and filter out ad banners and other unwanted images. For logging in, I utilized an authentication librarycalled PassportJS to enable the creation of accounts via email, facebook, twitter, or google. Information about the recipes are stored in a database as JSON format in a cloud MongoDB and storage of pictures are stored in amazon S3.

Tic Tac Toe

I made this to see how far I can push my algorithm skills. Because the number of moves are limited, the algorithm is quite simple. There are 4 main moves the computer can take. An offensive move, where the computer can determine where to place to beat the game. A defensive move, where the computer can determine where to place to prevent the user from winning. The last 2 moves are predetermined moves where it will always guarentee a win based on the starting position of the user.
